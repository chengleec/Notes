#### 希腊字母

要输入希腊字母只要用反斜杠 `\` 加上相应字母的拼写即可。大写字母将对应拼写的首字母大写。

| 符号       | 语法     |
| ---------- | -------- |
| $\alpha$   | \alpha   |
| $\beta$    | \beta    |
| $\gamma$   | \gamma   |
| $\theta$   | \theta   |
| $\delta$   | \delta   |
| $\omega$   | \omega   |
| $\rho$     | \rho     |
| $\sigma$   | \sigma   |
| $\epsilon$ | \epsilon |
| $\lambda$  | \lambda  |
| $\phi$     | \phi     |
| $\psi$     | \psi     |
| $\mu$      | \mu      |
| $\nu$      | \nu      |
| $\eta$     | \eta     |
| $\zeta$    | \zeta    |
| $\xi$      | \xi      |
| $\pi$      | \pi      |
| $\tau$     | \tau     |
